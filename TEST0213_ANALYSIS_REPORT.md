**속도 보너스**:
- ≤ 1초: +5 XP
- ≤ 2초: +4 XP
- ≤ 3초: +3 XP

- ≤ 8초: +1 XP

**콤보 보너스**:
- 10연속 이상: +5 XP
- 7~9연속: +3 XP
- 5~6연속: +2 XP

### 2. XP 차감 (오답)

**기본 페널티**: `-(4 + Book)`

**연속 오답 배수**:
- 연속 3번 이상: 2.0배
- 연속 2번: 1.5배
- 처음 틀림: 1.0배

### 3. 레벨 진화

**레벨 업**:
- XP ≥ Lesson XP (`2 + Book`) → 다음 Lesson
- Lesson 25 초과 → 다음 Book, Lesson 1

**레벨 다운**:
