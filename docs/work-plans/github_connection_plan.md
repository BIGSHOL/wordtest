- **일시**: 2026-02-10
- **목표**: 로컬 작업 공간(`f:\wordlvtest`)을 GitHub 저장소(`https://github.com/BIGSHOL/wordtest`)와 연결

## 작업 내용
1. **GitHub 원격 저장소 추가**:
   - `origin`이라는 이름으로 `https://github.com/BIGSHOL/wordtest`를 추가 완료하였습니다.
2. **연결 확인**:

   - `git remote -v` 명령으로 연결 상태를 검증하였습니다.

## 향후 계획
